starter package

timeline:

2 〰 3 semanas

para quién es?

Este paquete es perfecto para quien recién a empezado con su negocio y no está listo para invertir en una experiencia de marca completa. Incluye elementos básicos de una identidad de marca que te harán destacar entre tus competidores y atraer a más clientes.

incluye:

  • Moodboard
  • Logotipo principal
  • Paleta de colores
  • Manual de marca

medium package

timeline:

4 〰 5 semanas

para quién es?

Este paquete es ideal tanto para empresas nuevas como para aquellas que llevan un tiempo en el mercado pero que creen que su marca ya no les funciona o no les representa. Con este paquete, elevará su marca a través de un diseño estratégico e intencional que trabaja para atraer a sus clientes ideales.

incluye:

  • Starter package +
  • Logotipo secundario
  • Imagen para redes sociales
  • Mockups para la marca

premium package

timeline:

6 〰 7 semanas

para quién es?

Si desea lucir profesional y comenzar a mostrar tu marca con confianza, este paquete es para tí. La característica principal de este paquete es la estrategia de marca. Profundizaremos para descubrir a quién exactamente a quién se dirige con su negocio.


incluye:

  • Starter package +
  • Medium package +
  • Tarjetas de visita/fidelización
  • Diseño de postal
  • Banners para redes sociales
  • Plantillas para redes sociales
  • Iconos para destacados de IG
  • GIF's IG
